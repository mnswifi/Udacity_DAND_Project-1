Introduction¶

In this project, we would be investigating a movie dataset covering a study period. This data set contains information about 10,000 movies collected from The Movie Database (TMDb), including user ratings and revenue. Certain columns, like cast and genres, contain multiple values separated by pipe (|) characters. There are some odd characters in the cast column. The final two columns ending with _adj show the budget and revenue of the associated movie in terms of 2010 dollars, accounting for inflation over time.

# function to plot average graph pattern corresponding to budget and revenue
def plotavg(x):
    '''
    Plotting the pattern of average revenue and revenue_adj, budget and budget_adj using line graph grouped by year.
    
    '''  
    #plotting graph
    ax = df_moviedata.groupby("release_year")[[x,x + "_adj"]].mean().plot(kind="line", figsize=(14,8),fontsize=18)
    ax.set_title("Line Graph of patterns in {} and {}\n".format(x,x +"_adj"),fontsize=22)
    ax.set_xlabel("\nRelease Year",fontsize=14);
    ax.set_ylabel("AVERAGE "+ str.upper(x) + ' $',fontsize=14);
    ax.legend(fontsize=16)

# Revenue against popularity.

df_prop = pd.DataFrame(df_moviedata.groupby(['release_year'])[['revenue','budget','popularity','runtime']].mean())

def Rplot(a,b):
    '''
    Plot to see the relationship between the most common features that influences a movie in respect to revenue.
    '''
    plt.figure(figsize=(8,5))
    plt.scatter(x=df_prop[a],y=df_prop[b]);
    plt.xlabel(a,fontsize=18)
    if b == 'revenue':
        plt.ylabel(b + ' $',fontsize=18)
    else:
        plt.ylabel(b,fontsize=18)
    plt.title(f"Distribution of {a} in respect to {b}\n",fontsize=22);
    


if __name__ == "__main__":
    Rplot('budget','revenue')

Conclusions¶

Nothing is constant and change is inevitable as shown by our line graph of budget and budget_adj, likewise revenue and revenue_adj. There exists irregularities from 1960 to 1980 in both comparison of budget and revenue to adjustments due to inflation in dollar. Factors like popularity, budget are keys to determine revenue from movie, but runtime has no significance to determine revenue generation as weak negative or no correlation was observed in our scatter plots used to test relationship. Lastly, a table showing most populous genres from year to year of study set covering 1960 to 2015 was obtained using max value function filtering algorithm to merge to corresponding genre as shown using a table.

Limitations¶

1) Some years had more movies releases than other years. In such case, rating on a yearly bases seems biase as they not evenly proportioned.

2) Genre should be properly streamlined to avoid ambiguity.
